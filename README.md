# CMD-coin_flip
DESCRIPTION:


This program acts as a game that replicates when a person is to flip a coin to make a decision.

Note: This is all ran within the command line

GAMEPLAY:


1. User is first asked to choose between heads or tails
2. After user chooses, CMD-coin_flip will randomly choose heads or tails
3. Once "coin" is revealed, User is asked if they would like to play again
4. If User says yes, steps 1 - 3 will be repeated until user indicates no on step 3
5. When user says no, their score of correct/incorrect answers will appear along with all coin-flip outcomes
6. End


Enjoy!
